# PerseusX test by Ronald Miranda
___

## Install instructions
1. Clone repository
```sh
git clone git@github.com:rmirandasv/perseusx-test.git
```
2. Install dependencies
```sh
cd perseusx-test
npm install
```
3. Start dev server
```sh
npm run dev
```
